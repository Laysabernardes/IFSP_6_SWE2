# Sistema de Gerenciamento de Produtos

Este projeto é uma aplicação web desenvolvida em **ASP.NET Core MVC** para a disciplina de **Sistemas Web II**. 
O objetivo é permitir o **gerenciamento completo de produtos**, com operações CRUD (Create, Read, Update, Delete) e exibição de detalhes, aplicando os conceitos de arquitetura MVC e persistência de dados com **Entity Framework**.

A aplicação foi criada para demonstrar o domínio das tecnologias **ASP.NET MVC**, **Entity Framework** e **Bootstrap**, focando em uma interface simples, responsiva e funcional.

## Funcionalidades

- **CRUD completo:** Crie, visualize, edite e exclua produtos.
- **Página inicial:** Lista todos os produtos cadastrados de forma organizada.
- **Detalhes do produto:** Exibe informações completas de um produto específico.
- **Validação de dados:** Garante o preenchimento correto dos campos obrigatórios e valores válidos.
- **Interface moderna:** Layout limpo e responsivo, com cores inspiradas na identidade visual do IFSP.
